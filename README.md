# Anthony's Custom Framework 16 RGB Macropad

This is the code for the daemon running in userspace that communicates with the QMK firmware running on my Framework 16's RGB macropad.

My fork of Framework's QMK firmware can be found at [AnthonyW2/qmk_firmware_fw16](https://github.com/AnthonyW2/qmk_firmware_fw16), with a list of major features in the readme.
